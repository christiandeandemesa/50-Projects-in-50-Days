# 3D Background Boxes

-   Displays a GIF within combined or seperated boxes. Built using HTML5, CSS3, SASS, and JavaScript. Other features utilized include a custom CSS reset and Google fonts.

## How to run this project

1. Open your terminal and type: git clone https://github.com/christiandeandemesa/50-Projects-in-50-Days.git
2. cd into the "50 Projects in 50 Days" folder, then cd into the "3D Background Boxes" folder.

## Features

-   User can view a GIF within a box.
-   User can click the button to spread out or combine the boxes.
-   Responsive web design for the Galaxy Fold, Samsung Galaxy S8+, iPhone SE, iPhone 12 Pro, Pixel 5, Samsung Galaxy A51/71, Samsung Galaxy S20 Ultra, iPhone XR, Surface Duo, iPad Mini, iPad Air, Surface Pro 7, Nesthub, and Nesthub Max in portrait and landscape sizes, and most standard laptops.
